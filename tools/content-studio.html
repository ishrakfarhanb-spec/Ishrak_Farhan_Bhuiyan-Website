Content Studio

Shape, stage, and ship your site content.

This workspace handles your blogs, news, and project library in one place. It is designed to feel less like a backup admin page and more like a real editorial desk: a searchable library, structured form editing, direct file output, and a live preview while you work.

Workflow

  1. Run open-content-studio.cmd.
  2. Connect this repo folder after the browser opens.
  3. Search or select an entry, then edit the working list.
  4. Use direct save when available, or download the active file.
  5. Run publish-site.cmd when you want the update live.
